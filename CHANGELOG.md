# 变更日志

本文档记录 `Schematica-survival` 的主要功能变更与行为调整。

## [0.1.0] - 2026-03-30

### 新增
- 新增 `schematica_printer` 功能方块，并注册对应方块物品与合成配方（铁锭 + 红石）。
- 新增打印机命令：
  - `/schematica printer print <x> <y> <z> [replace|solid|nonair]`
  - `/schematica printer provide <x> <y> <z> <itemId> <subtype> [count]`
- 新增打印机 GUI 改造：
  - 增加 `Load Outline`（确认键加载投影）
  - 增加 `Undo`
  - 增加 `Rotate / MirrorX / MirrorZ`
  - 增加材料供给区域（每种材料一个按钮）
  - 增加可拖动进度条/滚动条，支持滚轮与方向键滚动
- 新增帮助命令行 `help.line9`，展示打印机相关指令。

### 改动
- 打印机打印不再直接消耗玩家背包材料，改为消耗打印机方块库存。
- 打印机方块升级为带 TileEntity 的容器方块，破坏时会掉落内部物品。
- 材料统计逻辑统一到通用库存路径（玩家背包与打印机库存共用检查/消耗流程）。
- 客户端命令别名匹配增加 `schematica printer` 路径。
- GUI 与命令提示文本进一步键化，补齐中英文语言键。

### 打印行为与安全规则
- 新增“冲突阻止打印”：
  - 当投影目标位置存在非空气方块，且与目标方块（含 meta）不一致时，打印会被拒绝。
  - 聊天提示会返回第一个冲突坐标与双方方块信息。
- 新增“同块跳过”：
  - 若世界方块与投影方块一致，则该位置不重复放置。
  - 该位置不计入材料需求/消耗。
- GUI 供给面板需求量会在投影加载后按当前世界状态重算，自动扣除已完成位置。
